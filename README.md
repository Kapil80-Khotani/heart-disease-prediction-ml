# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview

This project builds an end-to-end Machine Learning pipeline to predict whether a patient is at risk of heart disease based on clinical and demographic features.

The project covers the complete Data Science workflow—from data exploration and preprocessing to model training, evaluation, and prediction.

---

## 🚀 Project Highlights

* Comprehensive Exploratory Data Analysis (EDA)
* Data Cleaning & Preprocessing
* Feature Engineering
* Handling Missing Values
* Outlier Analysis
* Feature Selection
* Multiple Machine Learning Models
* Hyperparameter Tuning
* Model Evaluation using multiple performance metrics
* Final Model Selection

---

## 📊 Dataset

The dataset contains patient medical information such as:

* Age
* Gender
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression (Oldpeak)
* ST Slope
* Number of Major Vessels
* Thalassemia
* Target (Heart Disease)
